📚 Summary

제 포트폴리오를 작성하면서 파일로 만들기는 아쉬워 직접 포트폴리오 웹을 만들어보고자 제작하게 됐습니다.
아무래도 front-end 경험이 전무한 상황에서 제작하려다보니 HTML, CSS, JavaScript에 실습을 조금 해보고, Bootstrap 템플릿을 통해 바로 제작을 진행하였습니다.
HTTP 통신없이 구현하였지만, 생각보다 신경써야 할 부분이 많아서 쉽지는 않았습니다.

📑 Meaning

평소 경험이 없던 부분을 처음으로 사용해본 프로젝트입니다. 처음에는 React 프레임워크를 사용하여 개발하려 했으나, 단순히 http 통신없이 정적 호스팅으로 충분할 것 같아 서버 리스의 구조로만 개발했습니다.
다만 사전지식이 충분하지 않기에 bootstrap 템플릿을 사용하여 구성하였습니다.
css를 설정하면서 각 컴포넌트 마다 직접 명시를 해줘야 하는 작업이 정말 헷갈리고 귀찮았지만 그래도 완성되어 가는 모습을 보며 뿌듯함을 느끼게 되었고, front-end 분들의 고충도 한츰 더 알게 되었습니다.
배포는 AWS의 S3 버킷 정적 웹사이트 호스팅을 이용하였고, 도메인을 구매하여 ACM에서 SSL 인증서를 받아 CloudFront를 통해 https로 배포 하였습니다. 이후 Route 53을 통해 제가 구매한 도메인으로 호스팅하였습니다.
마지막으로 해당 프로젝트에 대한 생각을 덧붙이자면 추후 서버리스의 구조가 아니라 작성자 권한의 로그인 기능을 넣어 글을 작성하여 DB에 넣을 수 있는 웹으로 리팩토링 하려합니다.
